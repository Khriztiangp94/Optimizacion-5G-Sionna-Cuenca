# Optimizacion-5G-Sionna-Cuenca

Optimización de Planificación de Redes 5G mediante Gemelos Digitales y Ray Tracing

Este proyecto de investigación se centra en el desarrollo de simulaciones de propagación para redes 5G, utilizando Gemelos Digitales (Digital Twins) para modelar entornos urbanos y optimizar el despliegue de infraestructura Massive MIMO mediante Optimización en 2 etapas, utilizando algoritmos como PSO y Bayesian.

Para el desarrollo se utiliza un entorno virtual con las siguientes versiones:
Núcleo de Simulación y Deep Learning
    Sionna / Sionna RT: 1.1.0 (Motor de Ray Tracing y capa física).
    Mitsuba: 3.6.2 (Renderizado y manejo de escenas 3D).
    TensorFlow: 2.19.1 (Framework de aceleración por GPU).
    Dr.Jit: 1.0.3 (Compilación JIT para simulaciones físicas).

Procesamiento y Optimización

    [Bayesian Optimization.](https://github.com/bayesian-optimization/BayesianOptimization/)
    PySwarms: 1.3.0
    NumPy: 1.26.4.
    SciPy: 1.15.3.
    Pandas: 2.3.3.
Las versiones y todas las librerias utilizadas se pueden consultar en el archivo requirements.txt
